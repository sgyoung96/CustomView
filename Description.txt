1. PinNumber : 

초기화면 구동 시 focus 가 되면서 시스템 숫자키패드가 자동으로 올라온다.
입력 값의 수 만큼 별이 노란색으로 변한다.
미리 지정해진 비밀번호 "123456" 을 입력하면 자동으로 화면이 전환된다.

2. RandomKeypad :

랜덤 버튼을 누르면 숫자 키패드의 숫자가 랜덤으로 바뀌게 된다.
삭제 버튼을 누르면 화면에 나타난 문자열이 하나씩 지워진다. (숫자 형의 문자열)
숫자 키패드를 누르면 ripple 효과가 나타난다.

!! Random 함수를 사용하여 중복제거를 하려고도 해 보았고,
Set 을 사용하여 중복제거를 하려고도 해 보았지만,
결국은 최악의 경우 - 0이 계속 할당되는 경우라면 무한루프에 빠질 가능성 + 처리 속도가 길어질 가능성이 있다.
